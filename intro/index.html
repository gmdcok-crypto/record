Why Hybrid

AI만으로 끝내지 않고, 사람이 책임지는 녹취 프로세스

완전 수작업은 시간이 오래 걸리고, AI 단독은 사건·민원·회의 문맥에서 놓치는 부분이 생길 수 있습니다. 저희 시스템은 속도와 정확도를 함께 잡기 위해 AI와 속기사 검수를 하나의 흐름으로 연결했습니다.

Comparison

AI 단독, 사람 단독보다 하이브리드가 실무에 강합니다

초벌은 AI가 빠르게, 완성도는 사람이 정교하게. 실제 제출용, 보관용, 검토용 문서에 필요한 균형을 맞춥니다.

AI 단독

  • 속도는 빠르지만 문맥 오인식 가능
  • 화자명, 끊어읽기, 법률·상담 표현에 취약
  • 최종 제출용 문서로는 재검토가 필요

AI + 속기사

  • 초벌 자동화로 작업 시간을 단축
  • 속기사 검수로 문장 완성도와 신뢰도 확보
  • 편집, 이력, PDF 결과물까지 하나로 연결

사람 단독

  • 정확도는 높지만 초기 작업 시간이 큼
  • 대량 파일이나 반복 업무에서 비효율 발생
  • 자동 요약·구간 탐색 같은 도구 활용이 제한적

Core Features

현재 구축 중인 녹취 시스템의 핵심 기능

실제 작업 흐름을 기준으로 업로드, 초벌 인식, 편집, 이력 관리, 문서 출력 기능을 하나의 시스템 안에 묶었습니다.

🎙️

다양한 음성 업로드

WAV, MP3, M4A, MP4 등 실무에서 들어오는 다양한 파일 형식을 받아 처리합니다.

🧠

화자 분리 AI 초벌

Soniox 기반으로 화자 구분과 초벌 문장 인식을 자동 생성해 초기 작업 시간을 줄입니다.

✍️

관리자 편집 화면

구간별 재생, 화자명 수정, 내용 보정, 저장 후 즉시 반영까지 편집 흐름에 맞춰 설계했습니다.

🗂️

수정 이력 관리

녹취록 저장 시 버전 이력을 남겨 이전 상태 복원과 변경 추적이 가능하도록 준비되어 있습니다.

📄

문서형 PDF 출력

표지, 끝표시, 페이지 푸터, 확인 문구를 포함한 문서형 PDF 결과물을 바로 생성합니다.

🔒

업무형 스토리지 구조

음성 원본과 녹취 결과를 분리 저장하고, 작업 단위 기준으로 결과물을 관리할 수 있습니다.

Workflow

의뢰부터 결과물 전달까지, 작업 흐름이 분명한 시스템

실제 실무자가 쓰는 순서대로 기능을 배치해, 단순 데모가 아니라 운영 가능한 녹취 프로세스를 목표로 합니다.

파일 접수

의뢰인이 파일을 업로드하면 작업 ID 기준으로 원본 음성이 저장됩니다.

AI 초벌 생성

화자 분리와 텍스트 초안을 생성해 속기사 검수의 출발점을 자동으로 만듭니다.

속기사 검수 및 보정

문장 흐름, 발화 구간, 화자명, 누락 표현 등을 기준으로 사람 검수를 진행합니다.

버전 저장 및 이력 관리

수정 이력을 남겨 작업 중간 단계와 최종 결과를 안정적으로 관리합니다.

PDF 결과물 출력

표지 포함 문서, 확인 문구, 푸터, 페이지 번호, 도장을 포함한 출력물을 제공합니다.

Output

단순 텍스트가 아니라, 문서로 바로 쓸 수 있는 결과물

결과물은 화면 확인용에 그치지 않고, 실제 전달과 보관을 고려한 PDF 문서 형식으로 정리됩니다.

  • 앞표지·뒷표지 포함 PDF 구성
  • 문서 끝 확인 문구와 종료 표시 반영
  • 페이지 하단 연락처, 페이지 번호, 도장 포함
  • 필요 시 이후 템플릿 확장도 가능한 구조
